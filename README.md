# github-practice

## 1. バグ報告Isuueを作成しよう

  サンプルプログラムsample.javaは実行するとエラーが発生します。以下は情報を含むIsuueを作成してください。

    ・タイトル
    ・発生した条件
    ・再現手順
    ・期待する動作
    ・実行環境

## 2. 追加機能Isuueを作成しよう

  サンプルプログラムCalc.javaには、加算機能int add(int a,int b)しかありません。減算機能を追加するためのIssueを作成してください。以下の項目を含むころ。

    ・機能概要
    ・利用者メリット
    ・完了条件
